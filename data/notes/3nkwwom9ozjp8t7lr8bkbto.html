Inat_data

Inat_data table (Figure 1) permits to store retrieved iNaturalist metadata. These metadata are retrieved using the iNaturalist API via the inat-fetcher script. With this, community curated taxons are updated every day in the database, permitting a continuous correction of the samples identity.
